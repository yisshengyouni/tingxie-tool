# 生字听写

一个简洁的在线生字听写小工具，支持语音朗读、调速、重复播放和分享链接。

## 功能

- **语音朗读**：使用浏览器内置语音合成朗读生词
- **自由配置**
  - 朗读速度：慢 / 中 / 快
  - 重复次数：1 ~ 3 遍
  - 间隔时间：2 / 3 / 5 / 8 秒
- **进度控制**：上/下一个、重播当前、停止播放
- **链接分享**：将生词和配置生成链接，发送给他人即可直接打开播放
- **微信兼容**：针对微信内置浏览器做了音频解锁和恢复优化

## 使用方式

1. 在输入框中填入生词，用逗号、空格或换行分隔
2. 选择朗读速度、重复次数和间隔时间
3. 点击**开始听写**即可播放
4. 点击**分享**按钮可复制链接，方便发送给朋友

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
